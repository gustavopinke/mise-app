# Pasta para fotos dos produtos

## Como adicionar fotos

1. Coloque as fotos dos produtos nesta pasta
2. Nomeie os arquivos com o código de barras do produto
3. Formatos suportados: .jpg, .jpeg, .png, .webp, .gif

## Exemplos de nomes de arquivo:
- `7891234567890.jpg`
- `7891234567890_mise.jpg`
- `7891234567890.png`

## Importante:
- O nome do arquivo DEVE começar com o código de barras
- O app buscará automaticamente a foto pelo código de barras
- Após adicionar fotos aqui, faça commit e push para o GitHub
